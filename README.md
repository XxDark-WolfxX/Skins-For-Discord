# Skins-For-Discord
These are custom themes for Discord users using BetterDiscord, LegCord, or Vencord.

If you find a bug, feel free to suggest an edit. If I don't respond in three days, feel free to email me @Dark_Wolf_Entertainment@outlook.com.

These are based on Microsoft's Windows 11 Fluid ui, and Fuvi from BetterDiscord. The credit for the idea goes to @FeoreV, as well as @Microsoft.

Expect updates to this repository from time to time.
